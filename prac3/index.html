Основи роботи з Git

  1. Відкрити папку з лабораторної роботи №2
  2. Ініціалізувати локальний репозиторій: `git init`
  3. Додати файли в репозиторій: `git add .`
  4. Закомітити файли: `git commit –m “IP labs init”`
  5. Прив’язати віддалений репозиторій: `git remote add origin “https://github.com/volzherebets/lab-pr.git”`
  6. Відправити коміти: `git push –u origin master`
